Cookies policy

What is a cookie?

Cookies are small amounts of information that are stored in the browser of each user so that the server remembers certain information that subsequently only the server that implemented it will read. Cookies generally function for a limited duration of time.

Which cookies do we use?

CRG uses the following cookies:

  • Technical cookies: They allow the user to browse a web page and to use the various options or services within, such as traffic control and data communication, session identification, access to restricted areas, etc.
  • Personal cookies: These allow the user to access the service with some predefined general characteristics such as the language, the type of browser used to access the service, the regional settings where access to the service is from, etc.
  • Analysis cookies: These allow tracking and analysis of user behaviour. The information collected by this type of cookie is used in the measurement of website activity and to develop user navigation profiles, in order to make improvements.
